TeamFinder
Платформа для поиска команд и проектов. Учебный проект для курса по Django.

О проекте
Проект позволяет:

Создавать и управлять проектами

Искать участников по фильтрам

Добавлять проекты в избранное

Управлять профилем пользователя

Технологии
Python 3.10+

Django 5.2

PostgreSQL (или SQLite для разработки)

Docker (опционально)

Bootstrap 5

Быстрый старт
Локальный запуск (SQLite)
Клонируйте репозиторий:

bash
git clone https://github.com/ssangbyte/team-finder.git
cd team-finder
Создайте и активируйте виртуальное окружение:

bash
python -m venv venv
source venv/Scripts/activate  # Windows
# или source venv/bin/activate  # Linux/Mac
Установите зависимости:

bash
pip install -r requirements.txt
Создайте файл .env (опционально, для переменных окружения)

Выполните миграции:

bash
python manage.py migrate
Загрузите тестовые данные:

bash
python manage.py seed_demo
Пользователи: 5 тестовых + суперпользователь. Пароль у всех: TeamFinder123

Запустите сервер:

bash
python manage.py runserver
Откройте в браузере: http://127.0.0.1:8000/

Запуск с PostgreSQL в Docker
bash
docker compose up -d --build

Демо-данные

5 тестовых пользователей (email: user1@example.com ... user5@example.com)

Суперпользователь: admin@example.com

Пароль для всех: TeamFinder123

У каждого пользователя есть по 2-3 проекта
